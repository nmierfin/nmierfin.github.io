Pupil Tracking Communication Software for Non-Verbal Individuals

I built a software application that would allow stroke patients suffering from complete paralysis, other than eye movement, to move a cursor on the screen using their eyes. The program allowed them to click and communicate on the screen using a virtual keyboard and prompts. This was done using linear adaptive filtering techniques using the coding language C++ in conjunction with OpenCV libraries on Visual Studio.
Site Service Plan for Chicane on Haist Street in Pelham

I drafted a site service pland for a chicane on Haist street in Pelham, Ontario. The purpose of the plan was to submit it to the city so that city council can determine whether it needed to be removed (due to multiple complaints). I used AutoCAD for the plan where I maintained scale and accuracy of the chicane as well as the surrounding areas.

PowerWorld Simulation for Solar Farm and Smart Grid Utilization

I designed a makeshift power grid utilizing the PowerWorld Simulator. This involved the sizing and design of the production, transmission and distribution (substation) of power to the customer. I had to size the solar panels needed for the production. In addition to this, I sized cables for three-phase transmission at 100kV, 500MVA. On top of that, I had to size transformers for stepping up and down voltage. Lastly, safety systems had to be put in place, such as lightning arresters, fuses, circuit breakers and relays to prevent faults in the larger grid.

UOE Supermilage

Designed the circuits for the peripheral system of an electric vehicle at the University of Ottawa. The peripherals that were designed include the steering, guage cluster (spedometer and tachometer), lights, turn signals, etc. The circuits for the perihperals were all custom designed on Altium. The circuits were all controlled via the Texas Instruments MSP430. The conrolling code was written in C as well as a RISC based ISA.

Laser Consolidation System

Designed the code for a metal power feeder and laser using Visual Basic on Visual Studio for the National Research Council. The system utilized a metal powder feeder, laser and motion setup for advanced 3d printing metal manufacturing. The program had to control paramters such as metal type, flow rate, laser intensity, etc. The system utilized g-code to move a cnc machine to draw out the manufactured item. I can't provide code for this system because it is proprietary to the NRC.

Card Counting Hi-Lo Method Simulator

I designed a hi-lo card counting simulator that would test the relative frequency of winning in blackjack for various amounts of decks and configurations. This would determine if the house or the player has the edge in regards to the methodology as well as deck configuration.
Home Alarm System

I coded a home alarm system in C that utilized the Freescale/Motorola HCS12. The program involved arming the system using a keypad. When the system is armed by typing in the passcode the alarm can be tripped by tripping a sensor. This will activate an interupt that causes an alarm to sound. The alarm can be disarmed by typing in the same passcode. It also utilized a 7 segment display that showed the passcode when typed.
